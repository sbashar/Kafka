## Commands

### Format HDFS filesystem

```bash
bin/hdfs namenode -format
```

### Start NameNode daemon and DataNode daemon

```bash
sbin/start-dfs.sh
```

### Stop NameNode daemon and DataNode daemon

```bash
sbin/stop-dfs.sh
```
