# C++
Notes on C++.

## Commands
Run normal C++ code using the following command:
    
    g++ -Wall hello_world.cpp -o hello_world
Important flags:
 * -Wall 
 * -W 
 * -Weffc++ 
 * -Wold-style-cast
 
 Run code with important flags:
    
    g++ -Wall -W -Weffc++ -Wold-style-cast hello_world.cpp -o hello.out
